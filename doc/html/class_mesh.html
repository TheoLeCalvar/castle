Detailed Description

Charge un modèle 3D.

Permet la lecture de fichiers contenant des modèles 3D

Warning
Ne supporte pas tout les formats, seul les .ply sont totalement gérés, les .obj le sont partiellement Ne lièbre pas (encore les vao/vbo)
